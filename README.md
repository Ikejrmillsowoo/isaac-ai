# Isaac AI

A private, local-first AI assistant built with:

- React
- Spring Boot
- Spring AI
- Ollama
- PostgreSQL
- pgvector

## Features

- Local LLM chat
- Document search (RAG)
- Workspace organization
- Conversation history
- AI-powered writing and research

## Tech Stack

Frontend

- React
- TypeScript
- Vite

Backend

- Spring Boot
- Spring AI

AI

- Ollama

Database

- PostgreSQL
- pgvector
